# Redrob Candidate Discovery Engine

A high-performance, memory-optimized screening and ranking pipeline engineered to parse massive talent datasets against structured job descriptions using context-aware lexical frameworks.

## 🚀 Key Features
- **Key-Agnostic Schema Processing:** Safely flattens variable candidate JSON records to prevent missing dynamic entries.
- **Lexical Vector Ingestion:** Leverages optimized TF-IDF representations to compute similarity spaces efficiently across massive corpora.
- **Hybrid Performance Matrix:** Implements a dual-weighted priority score combining semantic fit ($70\%$) and active platform footprint ($30\%$) metrics.

## 🛠️ Stack
- **Language:** Python
- **Core Libraries:** Pandas, Scikit-Learn, Python-Docx, Openpyxl
